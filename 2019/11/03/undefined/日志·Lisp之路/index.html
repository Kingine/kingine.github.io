日志·Lisp之路

What?

Lisp 编程模式是基于 lambda 演算的列表数据结构编程，使用波兰表达式(前缀表示)作为基本的表达形式。

七个原始操作符
1.(quote x) :简记 ‘x
2.(atom x) :原子 x
3.(eq x y) :x y 求值是否相同
4.(car x) :列表左子树
5.(cdr x) :列表右子树
6.(cons x y) :组合构造新表
7.(cond (p1 …e1) … (pn …en)) :依次求值，为true则求表达式的返回值

How?

SICP 这本魔法书中使用了该语言，每个使用该语言的人都是实至名归的魔法师。
方言: Common-lisp Scheme Elisp Autolisp clojure 等。
编辑器: Emacs DrRacket 等。
书籍: cl官网上大量的书籍 cl-book

Why?

其他的编程语言我实在是不太能弄懂，第一次接触 lisp 就很自然的理解其中的概念，不像其他语言中的语法糖那般迷惑。
学习编程这种事情，选自己喜欢的就好了。简单并非意味着容易，人生苦短，而我又愿意花十年时间去真正的了解 lisp。
